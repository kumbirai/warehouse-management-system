# Terraform Modules

Reusable Terraform modules for infrastructure components.

## Modules

- **kubernetes/** - Kubernetes cluster provisioning
- **database/** - Database provisioning (PostgreSQL)
- **messaging/** - Messaging infrastructure (Kafka)
- **networking/** - Networking components (VPC, Load Balancers)

Each module should be:

- Self-contained
- Reusable across environments
- Well-documented
- Versioned
